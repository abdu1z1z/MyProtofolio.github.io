AI Image Classifier

The Plant Classification Model is a deep learning project developed from scratch using SwiftUI for the front-end interface. The app allows users to upload images of plants, which are then classified into one of over 30 categories using a convolutional neural network (CNN). This model identifies the plant and provides users with detailed information, including the plant's name, description, and possible location within Saudi Arabia. The application was built from the ground up, with a focus on user-friendly design and seamless integration with machine learning capabilities. By leveraging SwiftUI, the app offers a smooth and interactive experience, making it easy for users to identify and learn about different plants. This tool is ideal for researchers, botanists, and nature enthusiasts looking for quick and accurate plant identification.

Data Analysis and bulid model

The Exploratory Data Analysis (EDA) Project focuses on analyzing forest fire data to uncover trends and factors that influence forest fire occurrences. The project involves a detailed examination of various weather conditions and fire-related indices such as temperature, relative humidity (RH), wind speed (Ws), rain, and other key metrics (FFMC, DMC, DC, ISI, BUI, FWI). These variables are studied to identify patterns and their relationship to fire occurrences. The analysis was conducted using Python, and includes data preprocessing, visualization, and machine learning techniques to predict forest fire risk. A Linear Regression model was used to predict the Fire Weather Index (FWI), with a focus on evaluating the model's performance using metrics like the Mean Squared Error (MSE) and R² Score, achieving an impressive R² score of 0.95, indicating a strong fit between the model and the data.

BallTracer Model

The BallTracer Model is a machine learning project designed to track and analyze the movement of a football (soccer ball) during a match. Using the YOLO (You Only Look Once) object detection model, the system identifies and tracks the ball in video footage, allowing the analysis of ball possession by different teams. The model detects players and ball positions on the field and provides insights such as the time each team has possession of the ball. Advanced features like tracking player speed, movement, and calculating distance covered during the match can also be integrated. This project is aimed at providing valuable data to football analysts, coaches, and enthusiasts.

The LoanStatus Project is a data science and machine learning project focused on predicting the status of loan applications. Using historical data, the model classifies whether a loan will be approved, rejected, or needs further review based on factors such as applicant income, credit score, employment status, and other financial indicators. The project utilizes supervised learning techniques, specifically classification algorithms like logistic regression or decision trees, to make predictions. This model can help financial institutions streamline their loan approval process and reduce risks by providing more accurate loan eligibility forecasts.
